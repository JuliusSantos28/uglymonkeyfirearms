# Ugly Monkey Firearms — Project Guide

## Quick Commands

- **Dev server:** `npm run dev` (serves at localhost:8080 with hot reload)
- **Build:** `npm run build` (outputs to `_site/`)

## Tech Stack

- **SSG:** Eleventy (11ty) v3 — generates static HTML from Markdown + Nunjucks templates
- **Styling:** Plain CSS with custom properties, mobile-first responsive design
- **Instagram feed:** Behold widget (free embed, added to gallery page once connected)
- **Contact form:** Formspree (free, replace `YOUR_FORM_ID` in contact.md once account is created)
- **Hosting:** Cloudflare Pages (auto-deploys from GitHub on push to `main`)

## Project Structure

```
src/                        # Eleventy input directory
├── _includes/
│   ├── layouts/base.njk    # Base HTML layout (head, nav, footer, mobile menu JS)
│   ├── nav.njk             # Navigation partial (active link highlighting)
│   └── footer.njk          # Footer partial
├── css/style.css           # All styles (custom properties, responsive, forms)
├── images/                 # Static images (passthrough copied)
├── index.md                # Home page
├── gallery.md              # Gallery page (Behold Instagram widget placeholder)
└── contact.md              # Contact page (Formspree form + contact info)
.eleventy.js                # Eleventy config (input/output dirs, passthrough copy)
```

## Adding / Editing Pages

Create a `.md` file in `src/` with front matter:

```md
---
layout: layouts/base.njk
title: Page Title
description: Meta description for SEO.
---

Your content here (Markdown or HTML).
```

## Deployment

- **Build command:** `npx @11ty/eleventy`
- **Output directory:** `_site`
- Push to `main` branch on GitHub → Cloudflare Pages auto-deploys
- Demo URL: `*.pages.dev` (auto-generated by Cloudflare)

## Styling Conventions

- CSS custom properties defined in `:root` for theming
- BEM-style class naming (`.block__element--modifier`)
- Mobile-first: base styles for mobile, `@media (min-width)` for larger screens
- `clamp()` for fluid typography
